### Prerequisites

- Node.js installed
- Git installed

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/rudalkunwar/recipe-app.git
   ```

2. Navigate to the project directory:
   ```bash
   cd recipe-app
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Run the app locally:
   ```bash
   npm start
   ```

5. Open in browser:
   - Visit http://localhost:3000
